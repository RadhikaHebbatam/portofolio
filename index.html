Profile

I spent the first decade of my career in production banking, healthcare, and logistics systems — building Web APIs, WCF services, and enterprise .NET backends that had to be right the first time, not just fast to ship. That discipline — clean architecture, CI/CD, structured logging, stakeholder alignment — is the same discipline I now bring to AI engineering.

At ASB Bank, I design and build Web API services for tech insurance products on the bank's production .NET stack, while prototyping applied-AI patterns — Semantic Kernel agents and Azure OpenAI integrations — that extend the bank's existing API-first architecture.

Outside my day role, I've shipped three end-to-end AI systems from scratch: a multi-agent appointment booking assistant, a citation-aware legal document RAG app, and a natural-language-queryable product catalogue API — each designed, built, deployed, and debugged solo, the same way I'd approach a production banking service.
